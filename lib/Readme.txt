Bibliothèques nécessaires au projet : 


-	JFreeChart et JCommon : Ces bibliothèques sont couramment utilisées pour créer des graphiques et des visualisations de données.

Objectif : JFreeChart est utilisé pour générer des graphiques et des diagrammes. JCommon est une bibliothèque de support pour JFreeChart.
Lien de téléchargement : Vous pouvez télécharger JFreeChart et JCommon à partir du site officiel : Téléchargement JFreeChart et Téléchargement JCommon.


-	MySQL Connector : Cette bibliothèque est nécessaire pour se connecter à une base de données MySQL depuis votre application Java.

Objectif : MySQL Connector est le pilote JDBC permettant la connexion à une base de données MySQL.
Lien de téléchargement : Vous pouvez télécharger MySQL Connector (MySQL JDBC Driver) à partir du site officiel de MySQL : Téléchargement MySQL Connector.


-	Passay : Cette bibliothèque est utilisée pour vérifier la complexité des mots de passe.

Objectif : Passay est une bibliothèque qui permet de vérifier la complexité des mots de passe, notamment en imposant des règles de force de mot de passe.
