Ressources nécessaires au projet :

- JBCrypt : JBCrypt est une bibliothèque de hachage de mots de passe qui permet de stocker et de vérifier les mots de passe en toute sécurité. Elle est utilisée pour sécuriser les mots de passe stockés dans la base de données.

  Description : JBCrypt est une bibliothèque Java open source qui implémente l'algorithme de hachage de mots de passe bcrypt. Il est couramment utilisé pour sécuriser les mots de passe des utilisateurs en les hachant de manière sécurisée, ce qui rend difficile leur récupération en cas de violation de données.
